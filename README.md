Diagnóstico de Segurança em Autocustódia

Esse site foi desenvolvido para que qualquer pessoa possa verificar o quão seguro (ou não) está o seu setup de autocustódia.

Ele faz perguntas-chave sobre a autocustódia do usuário e, logo após, mostra um "score", indicando se há risco ou não, quais são os pontos de vulnerabilidade e como o usuário pode resolver esses riscos.
